Beatriz is a Data Scientist and a R teacher at Curso-R. She is an R-Lady from São Paulo - Brazil.

Bea is a tidyverse instructor certified by RStudio, an international speaker (speaking at events such as RStudio Conference, Python Brasil, useR, and LatinR), and has written about programming in R in her blog since 2019.

In addition, she co-organizes communities and events, such as R-Ladies São Paulo, Latin-R, and satRday São Paulo.

Her favorite GitHub feature is GitHub Actions!
